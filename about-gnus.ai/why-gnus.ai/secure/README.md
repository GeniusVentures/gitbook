# Secure

Each application will include a Two-Factor Authentication system.

<figure><img src="../../../.gitbook/assets/Lock Icon.webp" alt="" width="188"><figcaption></figcaption></figure>

The system uses Time-based One-Time Passwords for verification, and the process is made more secure through a technology called Zero-Knowledge Succinct Non-Interactive Arguments of Knowledge (zk-SNARKs).
