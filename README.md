# Public Roadmap

* [ ] **March 2024**-  Official launch and release to the public.&#x20;
* [x] **December 2023**-  Customer portal complete.  Presale customers can now redeem or buy more GNUS tokens for A.I. and. M.L. processing.&#x20;
* [x] **October 2023**-  App user interface development.  App enters Beta phase.
* [x] **August 2023-**  App development including mobile wallet, mobile SDK for game integration, and a standalone app for desktops.
* [x] **2021-2022**-  Covid-19 pandemic and Russian/Ukrainian war hinder development.&#x20;
* [x] **June 2021-**  Minimally viable product (MVP) and Initial Coin Offering (ICO) allows for continual development.&#x20;
* [x] **May 2021**-  Dapps on Ethereum network complete and ready for Initial Coin Offering (ICO)
* [x] **April 2021**-  Hybrid blockchain development begins to implement fast blockchain with slow blockchain.
* [x] **October 2020**-  Coding begins on implementation of Directed Acyclic Graph (DAG) fast blockchain.
* [x] **January 2018**- Integration of blockchain into A.I. processing is started.  Sample game, shaders, and SDK development begin.&#x20;
